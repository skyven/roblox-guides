📊 变量与数据类型

Lua是一种动态类型语言，了解各种数据类型是编程的基础。

基本数据类型：

-- 数字类型
local age = 25
local score = 95.5

-- 字符串类型
local name = "Roblox"
local message = 'Hello World'

-- 布尔类型
local isOnline = true
local isAdmin = false

-- 表（类似数组/字典）
local playerData = {
    Name = "Player1",
    Level = 10,
    Items = {"Sword", "Shield"}
}

-- 空值
local nothing = nil

类型转换：

local num = 123
local str = tostring(num)  -- "123"

local text = "456"
local number = tonumber(text)  -- 456

🔧 函数与作用域

函数是组织代码的基本单元，理解作用域非常重要。

函数定义：

-- 基本函数
local function greet(name)
    return "Hello, " .. name .. "!"
end

-- 带默认参数的函数
local function add(a, b)
    a = a or 0
    b = b or 0
    return a + b
end

-- 可变参数函数
local function sum(...)
    local total = 0
    for _, num in ipairs{...} do
        total = total + num
    end
    return total
end

作用域规则：

  • local 变量只在当前作用域内有效
  • 全局变量不加 local 声明
  • 函数内部可以访问外部变量，但不能修改

🎯 事件系统

Roblox使用事件驱动编程，理解事件系统是开发的关键。

常用事件：

-- 触摸事件
local part = script.Parent

part.Touched:Connect(function(hit)
    local player = game.Players:GetPlayerFromCharacter(hit.Parent)
    if player then
        print(player.Name .. " touched the part")
    end
end)

-- 玩家加入事件
game.Players.PlayerAdded:Connect(function(player)
    print(player.Name .. " joined the game")
    
    player.CharacterAdded:Connect(function(character)
        print(player.Name .. "'s character spawned")
    end)
end)

-- 键盘输入事件
local UserInputService = game:GetService("UserInputService")

UserInputService.InputBegan:Connect(function(input, gameProcessed)
    if not gameProcessed then
        if input.KeyCode == Enum.KeyCode.Space then
            print("Space pressed")
        end
    end
end)

👨‍💻 面向对象编程

Lua通过表实现面向对象编程，掌握它可以编写更清晰的代码。

创建类：

-- 定义类
local Player = {}
Player.__index = Player

-- 构造函数
function Player.new(name, level)
    local self = setmetatable({}, Player)
    self.Name = name
    self.Level = level
    self.Health = 100
    return self
end

-- 方法
function Player:TakeDamage(amount)
    self.Health = self.Health - amount
    if self.Health <= 0 then
        self:Die()
    end
end

function Player:Die()
    print(self.Name .. " has died!")
end

-- 使用类
local player1 = Player.new("Warrior", 5)
player1:TakeDamage(30)
print(player1.Health)  -- 70

⚡ 性能优化

优化脚本性能可以提升游戏体验，以下是一些实用技巧。

优化技巧：

  • 减少全局查找：将常用服务存储在局部变量中
  • 避免重复计算：缓存计算结果
  • 使用合适的数据结构：选择正确的表类型
  • 限制渲染对象：使用LOD技术

优化示例：

-- 优化前：每次都查找服务
while wait(1) do
    game:GetService("Players"):GetPlayers()
end

-- 优化后：缓存服务引用
local Players = game:GetService("Players")
while task.wait(1) do
    Players:GetPlayers()
end

-- 使用task.wait代替wait
while task.wait(0.5) do
    -- 代码
end
💡 提示：

使用Roblox Studio的Profiler工具分析性能瓶颈，找出需要优化的代码。
